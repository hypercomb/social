# No YouTube Shorts

A Chrome/Edge Manifest V3 extension that passively removes YouTube Shorts.

It:

- Automatically activates visible **Show/View fewer Shorts** controls.
- Hides Shorts shelves, cards, and navigation links on desktop and mobile YouTube.
- Redirects direct `/shorts/VIDEO_ID` pages to the regular `/watch?v=VIDEO_ID` player.
- Can be enabled or disabled from its toolbar popup.

## Install

Open `chrome://extensions` or `edge://extensions`, enable **Developer mode**, choose **Load unpacked**, and select this folder. Reload any already-open YouTube tabs after installing.
