Help & User Instructions

Navigation

  • Left/Right Click Navigate between layers (shift+click to go back on Mac)
  • Mouse Wheel Resize the hive
  • Click & Drag Reposition the canvas

Tile Management

  • Tab Create new tiles on dark background
  • Tab
    Enter editing mode when hovering over tiles
    • Add captions, links, and images to tiles
    • Link to new layer via branch checkbox
    • Resize images with mouse wheel
    • Reposition images by dragging
    • Remove images via 'remove image' link

Media Features

  • Drag & Drop Drag images from desktop to create new tiles or change existing ones
  • YouTube Links Drag YouTube links onto canvas to add linked videos

Visual Cues

  • Tiles with wide, shaded borders indicate nested hives

Quick Actions

  • Esc Close this help page or cancel any operation
  • Click anywhere Close this help page
